## Track Sponsor Details

### Walrus
Verifiable Data Platform for AI and onchain finance. Provides storage, programmability, verifiability, and privacy. Core part of the Sui Stack (built by Mysten Labs). **WAL token** pays for storage and secures via staking.

Use cases:
- AI (verify training data / workloads)
- Data Markets (creators sell verifiable data)
- DeFi (real-time site & transaction authentication)

### DeepBook
Sui's on-chain orderbook & financial stack. Three primitives:
- **Spot**
- **Margin**
- **Predict** (binary markets, options, leveraged products — uses options oracle by Block Scholes)

Shared liquidity across the Sui ecosystem; one orderbook powers all integrations.

### Scallop (University Award sponsor)
Institutional-grade lending money market on Sui. TVL ~$21.58M. Features: trilinear interest rate model, veSCA, soft liquidation, flashloans, X-Oracle (powered by Pyth). Audited by Zellic, OtterSec, MoveBit, Asymptotic.
